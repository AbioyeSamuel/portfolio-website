Samuel Abioye is a full-stack web developer, software engineer and a first-class student of civil engineering pursuing a bachelor's degree.

I've had experience creating custom and interactive websites, web applications, database (both relational and non-relational) and software using several tech stacks, frameworks and libraries. My knowledge in this field is quite extensive.

In addition, I'm a team-player with an eye for detail, a team-leader and a problem-solver. I'm open to opportunities to implement my skills, knowledge and intellectual capacity.
